Nombre del programa:

Análisis y desarrollo de software

Descripcion:

Este programa te prepara para trabajar en el desarrollo y mejora de software, con
una formación completa en cómo crear y planear sistemas y aplicaciones.

Habilidades adquiridas:

Creación de aplicaciones para páginas web y teléfonos móviles. Conocimiento en
lenguajes de programación como Java, Python, JavaScript, entre otros. Diseño
y manejo de bases de datos. Capacidad para entender lo que necesita un sistema y
cómo debe funcionar. Pruebas de programas para asegurar su buen funcionamiento.
Trabajo colaborativo y organización de proyectos con métodos ágiles.
